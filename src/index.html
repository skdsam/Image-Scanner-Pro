Image Scanner Pro is a professional-grade image analysis toolkit built on Rust and Tauri. Beyond standard metadata extraction, it features a High-Precision Focus Peak analyzer, a Local OCR Engine for lightning-fast text search, and Dominant Palette Extraction with multi-format pro export (CSS, JSON, CSV).
